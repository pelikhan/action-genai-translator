Parámetros

Sección titulada «Parámetros»

La acción de GitHub pelikhan/action-continuous-translation admite los siguientes parámetros. Todos los parámetros son opcionales. Los valores predeterminados se muestran en la referencia a continuación.

Precaución

Esta acción está en desarrollo. ¡Los parámetros podrían cambiar en cualquier momento!

lang

Sección titulada «lang»

Tipo: string
Predeterminado: fr

Una lista de códigos ISO de los idiomas de destino, separados por comas (,), a los que se debe traducir la documentación desde el idioma de origen. Consulta una lista de todos los idiomas compatibles en el archivo models.mts en el repositorio.

source

Sección titulada «source»

Tipo: string
Predeterminado: en

El código ISO del idioma fuente en el que está escrito el contenido principal.

files

Sección titulada «files»

Tipo: string
Predeterminado: README.md

Una lista de archivos a procesar, separados por comas.

instructions

Sección titulada «instructions»

Tipo: string

Instrucciones adicionales para que el LLM las use al traducir.

instructions_file

Sección titulada «instructions_file»

Tipo: string

Ruta a un archivo que contiene instrucciones adicionales para que el LLM las use al traducir.

starlight_dir

Sección titulada «starlight_dir»

Tipo: string

Carpeta raíz de la documentación de Astro Starlight. Debe definirse si se define la opción starlight_base.

starlight_base

Sección titulada «starlight_base»

Tipo: string

Alias base para la documentación de Astro Starlight. Se utiliza para corregir los enlaces en las traducciones. Debe definirse si se define la opción starlight_dir. Actualmente no hay soporte para la opción sin base.

force

Sección titulada «force»

Tipo: boolean
Predeterminado: false

Forzar la traducción incluso si el archivo ya ha sido traducido.

debug

Sección titulada «debug»

Tipo: boolean
Predeterminado: false

Habilitar el registro de depuración. Lee más detalles en la documentación de registros de GenAIScript.

github_token

Sección titulada «github_token»

Tipo: string (se recomienda secreto)
Por defecto: ${{ secrets.GITHUB_TOKEN }}

Tu token de GitHub con al menos permiso de models: read. Lee más detalles en la documentación de Permisos de Modelos de GenAIScript en GitHub.
